---
title: "mouse2press"
description: "Simulates a right mouse button press."
---

```lua
mouse2press(x: number?, y: number?): void
```
Simulates a right mouse button press **only** if the roblox window is focused. Note that a press is distinct from a click, as you will need to release separately.

If `x` and `y` paramaters aren't empty, the mouse will move and press there.
