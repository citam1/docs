---
title: "mouse1press"
description: "Simulates a left mouse button press."
---

```lua
mouse1press(x: number?, y: number?): void
```
Simulates a left mouse button press **only** if the roblox window is focused. Note that a press is distinct from a click, as you will need to release separately.

If `x` and `y` paramaters aren't empty, the mouse will move and press there.
